题目描述

求有多少 $ n $ 位十进制数是 $ p $ 的倍数且每位之和小于等于 $ m_i (m_i = 0, 1, 2, \ldots, m - 1, m) $，允许前导 $ 0 $，答案对 $ 998244353 $ 取模。

数据范围 $n \le 10^9\ \ \ \ p \le 50\ \ \ \ m\le1000$
