CQOI2018滚粗记

发表于 2018-04-16

这真是一场奇怪的比赛。。。:shit:

Day 0

省选前一天恰好是女巫森林上线，中午开了一波包就去重邮了。雨下得比较大，所以樱花:cherry_blossom:什么的都是不存在的。穿过一片工地(为什么重邮里有类似居民区的工地？)去了那个贴了一个大二维码的大楼。里面环境还不错，就是电脑一般，键盘手感跟nk的一样，非常不错。试完机子就去吃饭了，我又找到了初中时喜欢去的那个小巷子，结果以前吃的店🥩 都没了？？？晚上看了回网络流建模，十一点拉闸睡觉。

Day1

早上起来睡得还算好，校门口随意买了几个包子吃了，8点15，进场

密码发了下来，-cs什么意思？看题，第一题一眼看到“原根”两字，当时就被吓得不轻，第一题就考数学，冷静下来看了题，不就是道BSGS板题，但本蒟蒻没写过BSGS，好在原理很简单，推了一下就写出来了。再看第二题，我一般不喜欢把题面拉下去，这个题面我看了三遍都没看懂什么意思，往下一翻，这个图怎么这么熟悉，不就是生成树计数？秒了之后就感觉非常不对劲，前两题都是板题是什么意思？再看第三题，这题模数怎么还是变化的，1e7的数据范围如何求逆？实在想不到快一点的做法，想到nlogn的算法不如交个暴力稳，敲了个$2^n$就去打对拍了

不知道过了多久，day1就这么考完了，心想岂不是GG?这场怕不是人均300，而我就拿了230，滚粗既视感，下午想在山上逛一圈，被老板憋回去了。大家都坚信网络流和数据结构是必考的，于是整个半天都在看大佬的网络流博客顺便玩一波战吼萨，动画放出来就看一道题，稳。老板把去年董神出的被切掉的题翻了出来，看了回标程没看懂，然后董神就押了一波题，明天会考简单计算几何、多项式裸题、不在树上的数据结构，于是看了回多项式，跟老van聊了一发就睡了。

Day2

早上很早就醒了，抱着翻盘的希望走进了考场，然而看到题心态爆炸了

T1 NOIP级别状压，T2傻逼递推，T3莫队板题（PPT上的那种）

省选题就出这种东西？？？

上了个厕所回来平复了下心态，心想这种题一定不能翻车，T1 卡卡常数0.5s稳，T3写个莫队压行不到800B，然而T2调到我爆炸。。矩乘+FFT高精度乘法跑得巨慢，卡了半天不行索性写了个O(n)的递推，卡卡常数极限数据能跑1.2s，比上面那个快了不知道多少。。。T1对拍半天写不出来，肉眼debug一发就没管了，看到斜前方唐老师正在扫雷。。。然而最后两分钟的时候，我突然想起T3要开long long！！！这时候dev突然打不开了！！！慌乱之中打开记事本加了个#define int long long 还没来得及编译，比赛就结束了！

考完出去非常爆炸，t2高精度没调对，t3可能会ce，唐老师他们表示很轻松。随便找了家华莱士吃饭，心情很不好，打了几把战吼萨，结果被脚本锤爆了。。。

到了出成绩的那一刻，480分，d1t1不知道为啥少了10分，d2t2递推还有60分，算了算总分比唐老师低10分，然后老板告诉我被挤出去了。。6号盆盆稳进队，巴蜀大佬AK，老van不知道为啥两道题报零了，老G表示省选AK也进不了队，kinda sad…话说董神押题可真准，t1简单计算几何，t2多项式(fft)，t3不在树上的数据结构6号的O(1)线段树
